# 第三方舆情 API 服务调研

## 📊 调研目标

找到性价比高、数据覆盖全、API 稳定的第三方舆情数据服务。

---

## 🔍 候选服务列表

### 1. 微热点（weihot.com.cn）

**覆盖平台：** 微博、微信、抖音、快手、今日头条、小红书等  
**核心功能：**
- 全网舆情监测
- 情感分析
- 热度追踪
- 报告生成

**价格：**
- 基础版：¥3000/月
- 专业版：¥8000/月
- 企业版：¥15000+/月

**API 能力：**
- 关键词搜索
- 文章详情获取
- 评论数据
- 传播分析

**优点：** 数据全面、情感分析准确  
**缺点：** 价格偏高、API 调用有限制

---

### 2. 清博大数据（gsdata.com.cn）

**覆盖平台：** 微信公众号、视频号、抖音、微博  
**核心功能：**
- 公众号文章监控
- 视频号数据
- 舆情预警
- KOL 分析

**价格：**
- 基础版：¥5000/月
- 专业版：¥10000/月

**API 能力：**
- 公众号文章检索
- 阅读量/点赞数
- 评论情感分析
- 传播路径

**优点：** 微信生态数据强、公众号覆盖全  
**缺点：** 其他平台数据较弱

---

- 内容排行榜

- 舆情监控
- 竞品分析

**价格：**
- 基础版：¥2000/月
- 专业版：¥5000/月

**API 能力：**
- 公众号数据
- 短视频数据
- 榜单查询

**优点：** 性价比高、短视频数据好

## 💡 推荐方案

### 方案 A：新榜基础版（性价比最高）
- **成本：** ¥2000/月 = ¥24000/年
- **覆盖：** 公众号、抖音、快手、小红书
- **适合：** 预算有限、重视短视频

### 方案 B：微热点基础版（数据最全）
- **成本：** ¥3000/月 = ¥36000/年
- **覆盖：** 全网平台
- **适合：** 需要全面监控

### 方案 C：混合方案（推荐）
- 新榜（抖音 + 公众号）：¥2000/月
- 自建微博爬虫：¥0
- 手动监控视频号：¥0
- **总成本：** ¥24000/年 + 服务器费用
